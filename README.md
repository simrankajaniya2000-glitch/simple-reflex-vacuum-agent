# Simple Reflex Agent for Vacuum Environment

## Overview
This project implements a Simple Reflex Agent in Artificial Intelligence to simulate cleaning behavior in a two-square vacuum environment.

## Objective
To demonstrate how an intelligent agent reacts based only on the current percept without considering past states.

## Problem Statement
The environment consists of two locations (A and B), which can either be clean or dirty. The agent must clean dirty squares and move efficiently between them.

## Features
- Detects dirty environment states
- Cleans squares automatically
- Moves between locations A and B
- Calculates performance score
- Tests multiple initial configurations

## AI Concept Used
Simple Reflex Agent

The agent follows condition-action rules:
- If current square is dirty → Clean
- Else → Move to next square

- Environment simulation with multiple test cases
